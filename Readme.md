# Server Log Monitoring & Generation Tool

### Опис
Це автоматизована система для генерації та аналізу серверних логів. Скрипт імітує роботу реального дата-центру, фільтрує критичні помилки та створює структуровані звіти.

### Основні можливості:
* **Автоматична генерація:** Створення до 1 000 000 логів за лічені секунди.
* **Smart Filtering:** Розумна фільтрація статусів `CRITICAL` та `ERROR`.
* **Dynamic Storage:** Автоматичне створення папок та іменування файлів за поточною датою.
* **Log Analyzer:** Вбудований парсер для підрахунку статистики інцидентів.

### Технології:
* **Python 3.x**
* **Modules:** `os`, `datetime`, `random`

### Як запустити:
1. Клонуйте репозиторій.
2. Запустіть файл: `main.py`
3. Перевірте папку `/logs` для перегляду результатів.
